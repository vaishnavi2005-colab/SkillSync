# SkillSync: Automated Task Orchestration Engine

## Overview

SkillSync is an intelligent task orchestration system that automatically assigns tasks to employees based on their skills, experience level, availability, and workload capacity. The system reduces manual task assignment effort and improves resource utilization through smart automation.

## Features

* Employee Management
* Skill Management
* Employee-Skill Mapping
* Task Creation and Tracking
* Smart Task Allocation
* Workload Balancing
* Capacity-Based Assignment
* Task Status Management
* Task Completion Tracking
* Dashboard with Statistics

## Technology Stack

### Backend

* Java
* Spring Boot
* Spring Data JPA

### Frontend

* HTML
* CSS
* JavaScript
* Bootstrap

### Database

* SQLite

## System Workflow

1. Create Employees
2. Add Skills
3. Map Skills to Employees
4. Create Tasks
5. Assign Required Skills
6. Run Smart Allocation
7. Automatically Assign Best Employee
8. Track Task Progress
9. Complete Tasks and Update Employee Statistics

## Smart Allocation Logic

The allocation engine selects employees based on:

* Required Skill Match
* Employee Availability
* Current Workload
* Maximum Capacity
* Balanced Task Distribution
